# AI‑Assisted Coding Exercise

This is a short, real‑world coding exercise intended to be solved live.

You may use **any AI coding assistant you normally use**
(GitHub Copilot, Copilot Chat, Claude, Cursor, etc.).

## Instructions

1. Review the function in `headers.py`
2. Run or reason about the failing test in `test_headers.py`
3. Fix the bug and explain the root cause
4. Extend the function per the feature request
5. Make reasonable design choices and explain them

We care about:
- Correctness
- Code quality
- How you reason and validate
- How you use AI as a tool

There is no single “right” solution.
